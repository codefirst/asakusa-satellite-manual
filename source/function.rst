部屋作成
-----------------------
1. トップページで Create New Room ボタンをクリックします

.. image:: images/create_new_room_top.png

2. 部屋名を入力し、Create ボタンをクリックします。

.. image:: images/create_new_room.png

3. 部屋が作成されます。

.. image:: images/new_room.png
